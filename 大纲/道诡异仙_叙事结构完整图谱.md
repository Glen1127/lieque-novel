## 四、主角认知弧线完整版（5阶段20章切）

```
阶段1：否认期（第0-10章）
  核心：这边是幻觉/那边是真实
  关键：物品跨世界确认（Ch2-3）→丹药有效（Ch5）→情感崩溃（Ch9）
  结果：必须二选一

阶段2：利用期（第11-30章）
  核心：利用丹方世界获取资源
  关键：谋略反击（Ch13-17）→丹阳子死（Ch21）→游老爷重召（Ch30）
  结果：发现两个世界都有真实联结

阶段3：崩溃与重建（第31-60章）
  核心：哪个是真的？我能信任谁？
  关键：腊月十八身份替换（Ch90）→正德寺信任崩塌（Ch45）
  结果：认知最深处的不确定

阶段4：身份确立期（第61-129章）
  核心：我是谁？我是不是怪物？
  关键：心素=唐僧肉（Ch66）→荒岛屠杀（Ch68-77）→屠杀真相（Ch125）
  结果：确认自己是加害者但选择继续

阶段5：接受与分裂（第130-242章）
  核心：即使不确定，即使付出代价，我选择活着
  关键：南天门真相（Ch172）→死亡复活（Ch174）→身体意识分离（Ch198）
  结果：开放式结局；身份永远分裂但有锚点
```

## 五、叙事节奏规律总结

### 核心规律（五条）

1. **每5章一个小波峰**：平淡章节后接意外事件
2. **每20章一个核心波**：大危机+反转+代价（10/10张力）
3. **每章必须有新信息**：新世界规则/新认知/新威胁
4. **代价必须当下显现**：不拖超过3章
5. **章末类型交替**：悬念型/事件型/认知转变型/高潮型 交替出现

## 七、结局分析

### 章节242的开放式结局

**最后场景**：李火旺被白穗蛇捕获，困在洞穴中。白灵淼在上方被拉着。他浑身是血，用最后几根手指，看着无法加入的人。

**这不是有答案的结局，而是确认主题的结局：**

1. **身份问题无解** — 两边世界都无法确认真实，但人依然选择继续
2. **代价持续累积** — 从缺指甲到缺手指到几乎丧命
3. **关系是唯一锚点** — 白灵淼/同伴/狗，是活着的理由
4. **世界永远危险** — 没有安全目的地

### 核心主题（一句话）

> **"即使不知道哪边是真的，即使付出一切代价，我依然选择继续活着。"**
