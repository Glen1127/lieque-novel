## 六、道诡异仙核心叙事引擎

### 叙事公式
```
已知真实（某世界） + 跨世界流通 + 精神不稳定 = 永续不确定性
+ 暴力威胁（黑鼎/丹阳子/外部邪物）+ 情感锚点（杨娜/母亲/白灵淼）= 生存动力
```

### 叙事节奏规律
1. **每5章一个小波峰**：在平淡章节后接一个意外事件
2. **每20章一个核心波**：一个大危机+反转+代价
3. **每章必须有新信息**：要么是新的世界规则、要么是新的认知、要么是新的威胁
4. **代价必须当下显现**：不能拖延超过3章

### 情感锚点设计
- **杨娜**：现实世界的情感锚点，她的存在让主角不能彻底放弃现实
- **白灵淼**：异世界的情感锚点，从"被救者"成长为"并肩者"
- **吕戏班**：人间的温暖，展示"正常生活"的可能性
