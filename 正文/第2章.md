# 第2章 迷雾

沈念回到档案室时，天色已经暗了下来。

走廊里的日光灯嗡嗡作响，空气中弥漫着纸张特有的霉味。他推开沉重的大门，发现今天的档案室比平时更加安静。几个同事已经下班，只有最里面的那盏灯还亮着，那是陈姐的办公室。

他犹豫了一下，没有直接去交报告，而是转身走进了档案室深处的阅卷区。

那里有一扇他以前从未注意过的门。

不对。沈念停下脚步。三个月前他刚来的时候就注意到了，只是那时候他以为那只是存放旧档案的储藏室。现在想来，那扇门的样式太新了，与周围斑驳的墙壁格格不入。

门是深灰色的，没有把手，只有一个圆形的凹槽，像是某种感应装置。

沈念掏出他的深层档案协理员证件，在凹槽前晃了晃。

滴。

一声轻响，门缓缓滑开。

门后是一条狭长的走廊，墙壁上嵌着柔和的壁灯，脚下是防滑的橡胶地板。这与外面的老旧档案室完全是两个世界。沈念犹豫了一秒，迈步走了进去。

走廊尽头是一间不大的房间，正中央摆着一张桌子，桌旁坐着一个人。

那是个看起来三十岁左右的女性，穿着深蓝色的工作服，头发束成利落的马尾。她的面前摊着一本厚厚的册子，手边放着一杯还在冒热气的茶。

“沈念，对吧？”女性抬起头，“我是你的引导者。你可以叫我林姐。”

沈念点了点头，没有说话。他在桌子对面的椅子上坐下，把证件放在桌上。

“第一天就去大剧院了？”林姐翻开册子，“动作挺快。”

“我同事的父亲在那附近失踪了。”沈念说，“我想查一下档案。”

“大剧院仓库。”林姐点了点册子上的一行字，“周德明，五十八岁，上个月初失踪。档案室记录在案。”

“档案室？”沈念皱眉，“他是被……转移走的？”

林姐没有立即回答。她端起茶杯，轻轻吹了吹，然后啜了一小口。

“沈念，你对'转移'这个词的理解是什么？”

“档案里有提到过这个词。”沈念说，“回声转移。单向不可逆。”

“没错。”林姐放下茶杯，“然而你可能不知道，回声转移是档案室的核心职能之一。我们不是随随便便带走任何人——每一个被转移的人，都已经死了。”

沈念的手指微微动了一下，然而他没有说话。

“你以为他们只是失踪。”林姐继续说，“家人以为他们离家出走，警察以为他们自己离开。然而实际上，他们已经在某个时刻死了。心脏病突发、意外事故，或者在错误的时间出现在错误的地点。”

“周小梅的父亲呢？”

“周德明。”林姐翻到册子的某一页，“上个月初，他在仓库整理货物的时候，遇到了一个醉酒驾车冲进仓库的司机。送来的时候已经救不回来了。”

沈念沉默了很久。

“那他的……回声呢？”

“死亡证明没有公开，因为某些原因，你不需要知道原因。总而言之，周德明的回声现在在深层，等待审判。”

“等待审判之后呢？”

“流转。或者消散。或者永远留下。”林姐说，“取决于他生前的行为。”

沈念终于理解了。档案室并非简单的失踪人口处理机构，而是阴阳两界的中转站。亡魂被带来这里，登记、转移、送往深层。而他，现在成为了这个系统的一员。

“为什么告诉我这些？”他问。

“然而同时。”她顿了顿，“你也有权利选择退出。

沈念低头看着他的证件。蓝色的卡片，边缘已经有些磨损。

“名单已满，是什么意思？”

林姐挑了挑眉，显然没想到他会问这个。

“深层不是无限开放的。”她说，“每个月只有十二个名额。十二个孔位，满了就要等下个月。如果有新亡魂进来，就必须有旧的先流转出去。这是规则。”

“如果不遵守呢？”

“系统会崩溃。”林姐简短地说，“到那时候，所有回声都会溢出——到阳间。”

沈念消化着这些信息。十五年的档案室工作让他学会了在信息过载时保持冷静，他会把它们分门别类，建立索引，找到关键点。

所以周德明确实死了。周小梅的父亲已经不在这个世上了。

他需要告诉周小梅吗？

这个念头刚冒出来就被他否定了。现在不是时候。并非他在乎承诺，而是他还没有完全理解这个系统的运作方式。贸然告诉一个失去父亲的人"你爸已经死了，灵魂被我们收走了"，只会造成更大的伤害。

他需要更多信息。需要理解整个系统，需要找到证据，需要确认，然后才能做出决定。

“大剧院仓库下面有什么？”他问，“我今天在那里听到了一种声音。很低沉的嗡鸣，从地下传来。”

林姐的表情变了。

“你听到声音了？”

“很明显。”沈念说，“就在仓库门后面。那下面有什么？”

林姐合上册子，站起身来。她走到墙边，按下一个按钮。一块嵌板滑开，露出一面显示器。

“深层入口。”她说，“并非常规入口，然而确实存在。大剧院仓库是早期的转移点之一，后来废弃了，然而设施还在运行。”

“转移点？”

“就是把回声从阳间带到深层的地方。”林姐说，“每个转移点都有自己的频率。你听到的嗡鸣，是设备运转的声音。”

“设备？”沈念皱眉，“什么设备？”

“这个问题超出了你目前的权限。”林姐说，“我只能告诉你：大剧院仓库的转移点已经废弃很久了，理论上不应该有任何活动。然而如果你听到了声音，说明它还在运行。”

“是谁在运行它？”

林姐没有回答。她转过身，重新坐下。

"沈念，我再说一遍。"她的声音变得严肃，"此事不应由你来管。你是新来的，你不知道深层的水有多深。周德明的例子不是个例——在过去三个月里，有十七个亡魂被转移，其中至少有五个是大剧院区域的人。"

“他们在那里被杀？”

“或者被选中。”林姐说，“有些亡魂活着的时候就有执念，死了也不肯散去。深层需要这样的回声，它们是维持系统运转的燃料。”

沈念感到一阵寒意。

“你是说……有人在故意制造死亡？”

“我没有这么说。”林姐重新端起茶杯，“然而我可以说：大剧院仓库不是一个安全的地方。如果你还想活着继续做你的档案工作，就离那里远一点。”

沈念沉默了很长时间。他在整理信息，把所有线索摊开，分门别类。

周德明死了，被转移到了深层。大剧院仓库地下有转移设备，还在运行。转移需要"燃料"，而燃料是亡魂。

这些线索之间有什么关联？

一个管理员，在仓库工作二十多年，无故失踪。他的女儿来上班，眼睛哭肿。

然后他发现了地下有设备在运转。

也许设备需要维护。也许周德明看到了什么。他也许并非"被选中"，而是"被处理"。

“我需要去那里。”沈念说。

林姐抬起头，眼神变得锐利。

“你没听见我说的话吗？”

“我听见了。”沈念说，“然而我同事还在等她的父亲。我不能告诉她真相，至少，我需要找到真相。”

“你知道代价是什么吗？”

沈念想到了他的证件上那句话。深层档案协理员。

“正常人资格。”他说，“对吗？从我接过这张证件的时候起，我就已经不是普通人了。”

林姐盯着他看了很久。

“你和你父亲真像。”她突然说。

沈念愣了一下：“你认识我父亲？”

“他在十年前也是档案室的人。”林姐说，“后来他选择了退出。他没有告诉你吗？”

父亲从未提起过这件事。在沈念的记忆里，父亲只是一个普通的工厂工人，早出晚归，沉默寡言，在三年前因病去世。

“看来他选择让你自己发现。”林姐说，“好吧。既然你决定了，我也不再劝你。然而有一个条件。”

她从抽屉里拿出一个小瓶子，里面装着几颗白色的药丸。

“如果你感觉到不对劲，”她说，“头痛、幻听，或者发现自己开始做奇怪的梦，立刻吃一颗，然后来找我。这些药可以延缓代价的侵蚀，却不能根治。”

沈念接过药瓶，放进口袋。

“谢谢。”

“别谢我。”林姐说，“我只是不想这么快就再换一个协理员。培养一个新人很麻烦的。”

沈念站起来，走到门口。他回头看了一眼，林姐已经重新翻开册子，开始记录什么。

“还有个问题。”他说。

“说。”

“名单已满——如果我想把周德明从深层弄出来，需要怎么做？”

林姐笔尖一顿。

“你想让他回到阳间？”

“只是问一下。”

“名额替换。”林姐说，“让一个已经流转的人把位置让出来。”她抬起头，“或者找到一个新的'孔'。”

“什么意思？”

“十二孔是明面上的规则。”林姐说，“然而如果有人能在深层找到第十三个孔位，就可以绕过限制。不过那需要极大的代价，不是谁都能做到的。”

沈念没有再问。他打开门，走出去。

走廊里的灯光仍然嗡嗡作响，然而他感觉整个世界都已经不同了。

明天是周六。档案室不上班。

他可以去大剧院仓库。

并非为了冒险——他知道自己不是那种冲动的人。他只是需要确认一件事。

周德明到底是意外死亡，还是被"处理"了。

如果是后者。

沈念握紧口袋里的药瓶，眼神变得深沉。

他需要让某些人付出代价。

即使那意味着他自己也会失去一些东西。

※

周六清晨，沈念骑着他的二手自行车，再次穿过三条主街和那段仍在施工的隧道。

天气仍然闷热，汗水浸透了他的衣领。然而这次他的目的地不再是未知，而是那个发出嗡鸣声的地方。

大剧院仓库。

铁门仍然锁着，然而这次他注意到了不一样的东西。

门锁是新的，然而门框上有新鲜的划痕——像是最近有人强行撬开过。

沈念绕到仓库侧面的小巷。巷子尽头是剧院的侧门，同样紧锁着，然而门前的地面上，有几个新鲜的脚印。

他蹲下来，仔细观察。脚印不大，应该是女性的尺码。

周小梅的母亲？

沈念站起身，环顾四周。清晨的巷子里空无一人，然而空气中弥漫着一股奇怪的味道——并非腐烂，而是一种类似于烧焦塑料的刺鼻气味。

嗡鸣声比上次更清晰了。

他深吸一口气，决定不再从正门进去。他记得上次看到的侧面窗户——虽然关着，然而玻璃已经老旧，也许可以撬开。

他需要找到真相。

即使代价是他再也无法回头。
