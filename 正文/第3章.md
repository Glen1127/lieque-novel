# 第三章 加班

第四天，沈念加班到很晚。

他本来应该下午六点下班的。但他把这个罐子拿出来看了三次。每次他都觉得自己应该把它放进"不要"的箱子里，但他没有。他不知道为什么。

七点半，整个三楼只剩他一个人。灯是感应灯，他走动的时候才会亮，他停下来的时候就灭了一半。他坐在货架之间的过道里，罐子放在膝盖上。

他在等什么？他不知道。

然后灯灭了。

不是感应的那种灭，是突然的，完全的灭。沈念坐在黑暗里，听着自己的呼吸声。他没有动。他等了几秒钟，等灯重新亮起来。

灯没有亮。

然后他听到了声音。

不是罐子里的声音。是从货架后面传来的，很轻，很远，像是什么东西在动。

沈念站起来，但没有走。他站在原地，看着货架之间的过道。黑暗里什么都看不见，只有声音。声音越来越多，越来越清晰。他听到了脚步声，说话声，还有什么东西在地上拖行的声音。

然后灯亮了。

他站在货架之间，手里拿着罐子，周围什么都没有。没有脚步声，没有说话声，没有东西在地上拖行。只有罐子里灰白色的粉末在手电筒的光线下呈现出那种奇怪的质地。

他看了看手表。八点十五分。

他在这里坐了多久？他不知道。时间过去了，但他没有感觉到。他只知道罐子里的声音变得更清晰了。不是罐子本身的声音，而是罐子里那个人的声音。

**"你在这里。"**

这次他听得很清楚。不是脑子里出现的声音，是真的听到了，像是有一个人站在他身边说话。

**"我等了很长时间。"**

声音停了。

沈念把罐子放回口袋里，拿起手电筒，下了楼。
