# 《道诡异仙》→《裂隙》骨架映射与意同形不同规则

> 本文件是《裂隙》写作的核心约束文档。context-agent 每次生成写作任务书时**必须加载此文件**，并将以下规则作为硬性约束注入任务书。
> 
> 禁用：直接复制《道诡异仙》情节/人物/设定，仅做表面替换。
> 要求：核心情感结构相同（迷失→代价→不可逆），具体情节/人物/世界维度完全不同。

---

## 一、意同形不同总规则

### 主角类型差异
| 《道诡异仙》李火旺 | 《裂隙》沈念 |
|------------------|-------------|
| 战斗型（硬冲、直接、情绪外放） | 整理型（先分类再行动、内敛、情绪内收） |
| 遇敌→战斗→突破 | 遇问题→分类→排序→归位→发现规律→再行动 |
| 用剑、丹方、修仙升级体系 | 用分类、档案、回声整理系统 |
| 可以靠实力赢 | 靠分类能力发现真相，不能靠拳头 |

### 世界维度差异
| 《道诡异仙》异界 | 《裂隙》异界 |
|----------------|-------------|
| 丹方（炼丹制药） | 回声（死者残留的声/记忆碎片） |
| 修仙有升级逻辑（炼气→筑基→金丹） | 档案系统无升级，只有"被记住"vs"没被记住" |
| 仙门/邪教/大梁组织 | 剧院/档案室/聆听者 |
| 丹方是资源，炼丹是目的 | 回声是原料，碾磨后归档/重新分配 |

### 情感锚点差异
| 《道诡异仙》 | 《裂隙》 |
|------------|---------|
| 杨娜=情感拯救者，温暖的光 | 陈姐=愧疚见证者+信息源，冷淡的旁观者 |
| 师傅=权威压迫，炼丹杀人 | 母亲=被处理的收集者，留下档案室的线索 |
| 丹方=利益目标 | 母亲的布片=请帖，不是工具 |
| 师姐/其他人=利益关系 | 悼=同类，另一个被处理的收集者 |

### 叙事技法差异
| 《道诡异仙》 | 《裂隙》 |
|------------|---------|
| 战斗场面多，打斗直接 | 整理动作多，发现真相靠分类 |
| 情绪外放，愤怒/恐惧直接表达 | 情绪内收，沉默/发抖/回避 |
| 信息通过对话直接输出 | 信息通过恐惧反应间接透露 |
| 穿越是身体在两个世界间切换 | 穿越是意识通过布片进入回声档案室 |

### 串线词（绝对禁止出现）
以下词汇出现说明情节已污染到《道诡异仙》原始设定，必须重写：
- 玄天宗 / 地下监牢 / 阿兄黑石 / 追兵 / 封灵索 / 玄棱场
- 筑基修士 / 反杀筑基 / 姜家岜峒 / 血脉觉醒者格杀勿论 / 外门弟子
- 丹方 / 炼丹 / 修仙 / 筑基 / 金丹 / 元婴 / 炼气

## 二、每章骨架映射

### 第1章骨架
**《道诡异仙》**：李火旺在料房工作，救白化病少女，癞子头师傅威胁他，被叫去见师傅，见证师傅杀人炼丹，穿越回精神病院。

**《裂隙》意同**：沈念在仓库整理货物，发现异常物品，遇到陈姐，陈姐恐惧躲避，沈念进入档案室发现母亲的相关记录。

**本章核心情感**：整理东西时发现世界有另一层秩序。

**禁止**：不能写沈念和人打斗，不能写他用武力解决问题。

---

### 第2章骨架
**《道诡异仙》**：精神病院现实世界，杨娜来探望送东西，李火旺开心，但医生警告他幻觉危险，他必须遵守那边逻辑才能出院。

**《裂隙》意同**：沈念在现实世界整理仓库记录，发现档案分类方法有规律，陈姐告诉他一些表面信息但实际上在隐瞒。

**本章核心情感**：发现整理能力不只是习惯，可能是某种更深的东西。

**禁止**：不能让陈姐直接告诉沈念真相，必须通过恐惧反应间接透露。

---

### 第3章骨架
**《道诡异仙》**：李火旺再次穿越回丹方世界，发现自己被师傅特殊对待，他决定调查师傅的秘密。

**《裂隙》意同**：沈念进入回声档案室，发现母亲曾在这里整理，悼作为同类出现，揭示母亲的选择。

**本章核心情感**：发现逝者留下的不是信息，是"等待被整理"的混乱。

**禁止**：悼不能是利益关系，必须是同类（另一个被处理的收集者）。

---

### 第4章骨架
**《道诡异仙》**：李火旺面对代价，师傅对他的控制加深，他开始质疑自己的处境。

**《裂隙》意同**：沈念逼问陈姐真相，陈姐崩溃交出钥匙，地下室门打开。

**本章核心情感**：打开禁忌之门之前，先要付出代价（情绪崩溃/信息泄露）。

**禁止**：沈念不能靠武力打开门，必须靠整理逻辑找到钥匙位置。

---

### 第5章骨架
**《道诡异仙》**：李火旺进入丹方深处，发现了更多真相，关于自己、关于师傅、关于这个世界的本质。

**《裂隙》意同**：沈念进入档案室，发现母亲的罐子，面临选择：转移母亲还是放弃。

**本章核心情感**：代价是必须做出选择，选择本身就是陷阱。

**禁止**：不能写沈念获得什么能力升级，只能写他"选择接受某个后果"。

---

### 第6章骨架
**《道诡异仙》**：李火旺做出了选择，发现选择带来更大代价，真相越来越可怕。

**《裂隙》意同**：沈念做出选择（接收母亲回声），母亲消失，沈念被标记，陈姐认出他。

**本章核心情感**：接受了某个东西就意味着失去另一些东西，被记住的代价。

**禁止**：沈念不能因为接收母亲变强，只能因为接收而被标记。

---

### 第7章骨架
**《道诡异仙》**：李火旺开始理解这个世界的规则，发现自己不是唯一清醒的人。

**《裂隙》意同**：沈念整理母亲记忆，发现规则第八条，用布片把母亲残影带回现实。

**本章核心情感**：规则存在的目的是让人知道边界在哪里，打破规则意味着失去保护。

**禁止**：布片不是工具，是母亲留下的请帖，沈念是被邀请进入的。

---

### 第8章骨架
**《道诡异仙》**：李火旺发现更深的真相，关于自己、关于组织、关于他在这个世界的位置。

**《裂隙》意同**：老周出场揭露舞台秘密，母亲被重新分配，小陶出场，沈念开始带小陶整理仓库。

**本章核心情感**：发现舞台背后的真相，但真相本身是更大的陷阱。

**禁止**：老周不能是反派推动者，必须是见证者+信息源。

---

### 第9章骨架
**《道诡异仙》**：代价开始显现，主角发现自己不是唯一一个"清醒的人"，真正的代价不是身体上的，是认知上的。

**《裂隙》意同**：小陶可能是母亲的新载体，沈念整理线索发现真相很可怕——他亲手把母亲送进了另一个人的身体。代价显现：被聆听者记住的代价是永远无法抹去做过的事。

**本章核心情感**：整理线索时发现自己的分类能力本身就是陷阱——他以为在调查真相，实际在帮聆听者优化档案。

**关键场景**：
1. 沈念发现自己手指发抖+回声低语（代价显现）
2. 小陶无意识使用母亲的整理方法（线索）
3. 老周揭示灰衣人=档案记录员（身份填坑）
4. 沈念拼出真相：小陶是母亲新载体（高潮）
5. 结尾钩子：聆听者在梦中揭示代价本质

**禁止**：
- 沈念不能直接冲出去质问，必须先分类整理再行动
- 小陶不能立刻知道自己身体里有母亲的回声
- 灰衣人不能变主要反派（只是执行者）
- 沈念不能快速接受现实，必须慢慢崩溃

---

### 第10章骨架
**《道诡异仙》**：主角面对更深的危机，发现自己只是更大棋局中的一子。

**《裂隙》意同**：沈念决定下一步行动——是否告诉小陶真相，如何处理与聆听者的关系。

**本章核心情感**：在"保持正常"和"继续深入"之间做出选择。

**待续**（本章需要根据第9章完成后实际情况调整）

## 三、世界设定映射

### 《道诡异仙》设定
| 设定项 | 内容 |
|-------|------|
| 异界 | 丹方世界，修仙体系，炼丹制药 |
| 核心组织 | 玄天宗（正派）、各种邪教 |
| 修炼等级 | 炼气→筑基→金丹→元婴 |
| 特殊能力 | 剑法、法术、丹药 |
| 终极真相 | 修仙世界是假的/被设计的，精神病院反而可能是真实的 |

### 《裂隙》对应设定
| 设定项 | 内容 |
|-------|------|
| 异界 | 回声档案室，档案整理系统，无升级逻辑 |
| 核心组织 | 剧院（舞台）、档案室（记录）、聆听者（深渊） |
| 能力等级 | 无等级，只有"被记住"vs"被碾磨" |
| 特殊能力 | 分类、排序、归位、感知回声 |
| 终极真相 | 回声被碾磨后归档，"被记住"意味着被写入某个更大的档案 |

### 核心设定对照
| 概念 | 《道诡异仙》 | 《裂隙》 |
|------|------------|---------|
| 另一个世界 | 丹方/修仙世界 | 回声档案室/剧院 |
| 穿越媒介 | 幻觉/精神不稳定 | 布片/请帖 |
| 核心规则 | 炼丹=升级=力量 | 分类=归档=被记住 |
| 代价 | 失去正常人资格/迷失 | 永远无法抹去做过的事 |
| 终极Boss | 不确定（可能是更大设计） | 聆听者（深渊本身） |
| 信息传播 | 对话直接输出 | 恐惧反应间接透露 |
| 拯救者 | 杨娜（情感温暖） | 无（陈姐是见证者非拯救者） |

## 四、悬念管理规则

### 活跃悬念池
每次写新章节前，从以下悬念池选择2-3个在本章推进，不能全部解决也不能全部不推进：

**长期悬念（3-5章填一次）**：
- 聆听者的终极目的——为什么要碾磨回声？回声最后去了哪里？
- 灰衣人的完整身份——档案记录员的权限边界在哪里？
- 舞台的秘密——这个规律是谁建立的？

**中期悬念（2-3章填一次）**：
- 母亲被重新分配去了哪里？
- 小陶是否是母亲的新载体？
- 老周凝视过聆听者吗？
- 布片空了之后还能触发穿越吗？

**短期悬念（本章填或引）**：
- 当前章节新埋的钩子

### 悬念推进规则
- 不能在同章节填掉3个以上悬念
- 每个活跃悬念最多跨5章不提及，否则读者遗忘
- 填坑必须付出代价（角色选择/信息泄露/关系变化）
