## 核心主题

**"到底哪边是真实的？"**

主角李火旺是一个精神病患者，能在现实（精神病院）和修仙世界之间穿越。随着故事发展，他发现这两个世界可能都是真实的，而他是一个特殊的存在：心素。

核心问题始终未完全解决，但主题通过李火旺的选择得到升华：

> **"即使不知道哪边是真的，即使付出一切代价，我依然选择继续活着。"**

## 叙事结构规律

### 节奏规律

1. **每5章一个小波峰**：平淡章节后接意外事件
2. **每20章一个核心波**：大危机+反转+代价
3. **每章必须有新信息**：新世界规则/新认知/新威胁
4. **代价必须当下显现**：不拖超过3章
5. **章末类型交替**：悬念型/事件型/认知转变型/高潮型 交替

### 认知弧线（6层恐惧升级）

```
第1层（Ch1-10）: "我有病吗？" — 精神病人身份
第2层（Ch11-30）: "我能杀死他吗？" — 复仇能否成功
第3层（Ch31-60）: "我能信任谁？" — 信任崩塌
第4层（Ch61-129）: "我是好人吗？" — 屠杀者身份
第5层（Ch130-242）: "我变成他了吗？" — 身份侵蚀
第6层（Ch201-242）: "即使不知道真假，我还活着吗？" — 接受与开放结局
```

## 开放式结局分析（Ch242）

**最后场景**：李火旺被白穗蛇捕获，困在洞穴中。白灵淼在上方被拉着。他浑身是血，用最后几根手指，看着无法加入的人。

**未解决问题**：
1. 心素能治愈吗？
2. 李火旺能否逃脱？
3. 二神对白灵淼的侵蚀会如何？
4. 坐忘道北风在哪里？
5. 大傩到底是什么？

**主题确认**：故事没有给出"哪边是真实的"的答案，但通过李火旺的选择确认了意义：即使不知道真假，即使付出一切代价，人依然选择活着。关系（白灵淼/同伴/狗）是唯一的锚点。

## 关键设定速查

### 世界结构
- **现实世界**：精神病院（现代）
- **异世界**：修仙世界（清风观/大梁/后蜀/青丘/梁国）

- **袄景教**：崇拜巴虺
